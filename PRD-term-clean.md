# PRD: term-clean

**产品名称**: term-clean
**版本**: v1.0
**文档日期**: 2025-01-27
**产品定位**: Mac 命令行包管理清理工具（TUI）

---

## 1. 产品概述

### 1.1 产品愿景
为 Mac 用户提供一个可视化的终端工具，用于管理和清理通过 brew（含 cask）、npm/pnpm/yarn、pip（含 --user）安装的命令行软件，解决"安装忘记、不知使用、不敢卸载"的痛点。

### 1.2 核心价值
- **发现与识别**: 一目了然查看所有已安装的命令行包
- **空间分析**: 精确计算每个包的磁盘占用，并展示依赖占用参考值
- **安全卸载**: 预览模式下展示影响，避免误删破坏系统
- **依赖可视化**: 分屏交互式展示依赖关系树

### 1.3 目标用户
- **主要用户**: 开发者（重度使用命令行工具）
- **次要用户**: 运维人员、DevOps、高级 Mac 用户
- **潜在用户**: 所有使用终端的 Mac 用户

## 3. 功能需求

### 3.1 MVP 功能（第一阶段）

#### F1: 查看列表
**功能描述**: 分类展示所有已安装的包

**详细需求**:
- 支持包管理器：brew（含 cask）、npm/pnpm/yarn、pip（含 --user）
- 分标签页切换显示
- 每个包显示：名称、版本、大小、最后修改时间
- 支持按名称/大小/时间排序
- 实时搜索过滤

**验收标准**:
- 能正确读取上述包管理器的安装列表
- 大小计算准确（以顶层包为准，依赖仅展示）
- 搜索响应时间 < 100ms

#### F2: 空间占用分析
**功能描述**: 计算并显示每个包的磁盘占用，依赖占用仅展示

**详细需求**:
- 主软件大小单独显示
- 依赖大小单独显示（仅展示）
- 总计大小汇总（仅顶层包）
- 共享依赖仅标注，不做可回收计算

**显示格式**:
```
ffmpeg
  ├─ 主软件: 50 MB (可清理)
  ├─ 依赖占用: 400 MB (仅展示)
  └─ 💡 可清理: 50 MB
```

**验收标准**:
- 大小计算误差 < 5%
- 支持人类可读格式（KB/MB/GB）

#### F3: 安全卸载
**功能描述**: 预览模式下显示将要删除的顶层包，依赖仅展示，确认后执行

**详细需求**:
- 显示将要删除的主包数量
- 显示依赖数量（仅展示，不删除）
- 显示将要释放的总空间（仅顶层包）
- 显示受影响的其他包（依赖此包的软件）
- 警告提示：可能导致的问题

**预览界面**:
```
🔍 Preview Uninstall

Selected packages (3):
  ✓ node@22
  ✓ ffmpeg
  ✓ python@3.12

Will be removed:
  • 3 main packages
  • Total: 750 MB

Dependencies (display only):
  • 15 dependencies

Affected packages (will remain):
  • npm (depends on node@22) ⚠️
  • Some tools depend on ffmpeg

⚠️  WARNING: Removing node@22 may break npm

[Confirm] [Cancel] [Export Plan]
```

**验收标准**:
- 依赖关系分析准确（用于展示与风险提示）
- 卸载失败时有回滚机制（仅顶层包）
- 提供卸载日志导出

### 6.3 核心模块

#### M1: PackageManager（包管理器抽象）
```python
class PackageManager(ABC):
    @abstractmethod
    def list_packages(self) -> List[Package]: pass

    @abstractmethod
    def get_package_info(self, name: str) -> PackageInfo: pass

    @abstractmethod
    def get_dependencies(self, name: str) -> List[Dependency]: pass

    @abstractmethod
    def uninstall(self, name: str) -> bool: pass
```

#### M2: DependencyAnalyzer（依赖分析器）
- 构建依赖关系树
- 检测循环依赖
- 识别共享依赖

#### M3: DiskUsageCalculator（磁盘使用计算器）
- 计算目录大小
- 区分主包/依赖
- 格式化显示

```python
@dataclass
class PackageSize:
    main: int  # bytes
    dependencies: int  # bytes
    exclusive_deps: int  # bytes
    shared_deps: int  # bytes
    total: int  # bytes

    def formatted(self) -> str: ...
```

```python
@dataclass
class Dependency:
    name: str
    version: str
    type: DependencyType  # EXCLUSIVE, SHARED
    dependents: List[str]  # 谁依赖我
```

## 12. 工程化规格（AI 开发指引）

> 目标：把本 PRD 落到可执行规格，便于 AI/研发直接实现。  
> 重要约束：**仅卸载“顶层软件包”，不清理依赖包**。依赖仅用于展示与风险提示。

### 12.1 作用域与边界（已确认）
- 仅支持全局安装（不包含虚拟环境/多版本管理器）
- Homebrew：包含 **Formula + Cask**，均支持展示/预览/卸载
- Node 生态：支持 `npm -g` + `pnpm -g` + `yarn global`
- Pip：包含系统全局 + `pip install --user`
- 卸载策略：**仅调用官方卸载命令**，不主动清理残留目录

### 12.2 数据采集与命令规范（仅顶层包）

#### 12.2.1 Homebrew（Formula + Cask）
- 列表：
  - Formula：`brew list --formula --versions`
  - Cask：`brew list --cask --versions`
- 详情：
  - Formula：`brew info --json=v2 --formula <name>`
  - Cask：`brew info --json=v2 --cask <name>`
- 大小统计：
  - Formula 目录：`$(brew --prefix)/Cellar/<name>/<ver>`
  - Cask 目录：`$(brew --prefix)/Caskroom/<name>/<ver>`
  - 使用 `du -sk` 统计，不扫描 `/Applications`
- 卸载：
  - Formula：`brew uninstall <name>`
  - Cask：`brew uninstall --cask <name>`

#### 12.2.2 Node 生态（npm / pnpm / yarn）
- npm：
  - 列表：`npm ls -g --depth=0 --json`
  - 根目录：`npm root -g`
  - 卸载：`npm -g uninstall <name>`
- pnpm：
  - 列表：`pnpm list -g --depth=0 --json`
  - 根目录：`pnpm root -g`
  - 卸载：`pnpm -g remove <name>`
- yarn：
  - 列表：`yarn global list --json`（逐行 JSON）
  - 根目录：`yarn global dir`
  - 卸载：`yarn global remove <name>`
- 大小统计：在各自 global 根目录下按包目录 `du -sk`

#### 12.2.3 Pip（系统全局 + --user）
- 列表：
  - 系统：`python3 -m pip list --format=json`
  - 用户：`python3 -m pip list --user --format=json`
- 详情：`python3 -m pip show <name>`（读取 `Location/Requires`）
- 大小统计：`Location` 下包目录 + `*.dist-info` 目录 `du -sk`
- 卸载：
  - 系统：`python3 -m pip uninstall <name>`
  - 用户：`python3 -m pip uninstall <name>`（仅当 Location 属于 user-site 时执行）

#### 12.2.4 统一数据字段
```
name, version, manager, scope(global/user/cask),
install_path, size_main, deps_for_display[]
```

### 12.3 依赖与空间计算规则（仅展示依赖）
- 依赖来源：
  - Brew：`brew info --json=v2` 的 `dependencies/optional/recommended`
  - Node：读取包目录 `package.json` 的 `dependencies`（深度=1）
  - Pip：`pip show` 的 `Requires`
- 依赖仅用于：
  - 依赖树展示
  - 风险提示（“被哪些顶层包依赖”）
- 空间口径：
  - **可清理空间 = 顶层包大小**
  - 依赖空间仅作为参考展示，不计入可清理空间
  - 共享依赖只做标记，不做可回收计算

### 12.4 异常处理与日志规范
- 未检测到包管理器：隐藏标签页并提示
- 权限不足：不自动提权，提示后允许重试
- 输出解析失败：降级显示基础信息并标注“数据不完整”
- 卸载失败：仅影响该顶层包，其他包照常处理
- 日志字段：时间、包名、管理器、命令、退出码、stdout/stderr

### 12.5 最小测试基线（MVP）
- 列表与解析：brew formula/cask、npm/pnpm/yarn global、pip global/--user
- 预览：被依赖提示、无依赖不提示
- 卸载：仅顶层包卸载，确认未删除依赖目录
- 异常：命令不存在、权限不足、输出异常、卸载失败、部分成功
